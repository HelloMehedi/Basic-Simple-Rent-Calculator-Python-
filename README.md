# Basic-Simple-Rent-Calculator-Python-
This is a simple Monthly Rent & Expense Calculator built using Python.
The program takes user input for house rent, daily food cost, utility bills (electricity, internet, gas), and number of persons, then calculates the total monthly expense and per person cost.
This project is designed for Python beginners to practice:

User input handling

Basic arithmetic operations

Functions

Error handling using try-except
